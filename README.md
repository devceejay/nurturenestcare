# NurtureNest - Childcare Website

NurtureNest is a modern, multi-page website concept for a fictional childcare center. Built with HTML, CSS, and JavaScript, it focuses on creating a warm, calming, family-centered user experience designed to build trust and engagement.

## 🌐 Live Demo:

Check the live site [here](https://nurturenest.vercel.app/)

##  📝 Overview:

This self-initiated project shows how a thoughtfully designed childcare website can balance playful visuals with professional structure. NurtureNest is aimed at parents exploring trusted early learning centers, combining informative sections, real-world UX patterns, and frontend interactivity while also hinting at how it could evolve with backend integration in the future.

## 📌 Features:

**Multi-page structure** - Home, About, Programs, Contact, Enrollment

**Programs page** - Cards with modals describing each program

**Meet the Team** - Highlights educators and staff to humanize the brand

**Daily Moments gallery** - Responsive photo grid showcasing daily activities

**FAQ accordion** - JavaScript-based toggle for common parent questions

**Parent testimonials** - Social proof section to build trust

**CTA banners** - Encourage parents to visit or enroll

**Fully responsive design** - Optimized for desktop, tablet, and mobile

## 💡 Planned Enhancements

While it’s currently frontend-only, I structured it to support future features like:

- Daily timestamp logs (mock attendance tracking)
- Calendar of events fetched from JSON (mock API)
- Admin login and dashboard UI prototype (to manage updates/content)

*These ideas show how the project could evolve into a real, dynamic platform.*

## 🔧 Technologies Used:

**HTML5** - Semantic layout and structure

**CSS3** - Custom styling, flex/grid layout, responsive design

**JavaScript** - Interactivity: modals, accordion, planned enhancements
